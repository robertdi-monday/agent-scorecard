# Agent Auditor — 60-Second Demo Script

> **Target time:** ~55–60 seconds at a calm, confident pace (~140 words)  
> **Audience:** Solution Architecture, Customer Success, Agent Builders

---

## THE SCRIPT

**[0–10s] — The Hook**

> "monday.com makes it incredibly easy to build AI agents.
> But once you've built one — how do you know it's actually *good*?
> Will it overspend your AI credits? Will it hallucinate and present it as fact?
> That's the gap Agent Auditor closes."

---

**[10–25s] — The Solution**

> "Agent Auditor is a 4-layer pipeline that audits *and* improves agents automatically.
> You point it at any agent. It tells you what's wrong. Then it fixes it."

---

**[25–50s] — The 4 Layers**

> "Layer 1: **Deterministic scans** — catches structural vulnerabilities in seconds.
> Layer 2: **LLM review** — checks whether the agent's instructions actually match its intended behavior.
> Layer 3: **Output verification** — compares what the agent *says* it did to what actually changed on the board.
> Layer 4: **Auto-fix generation** — for every issue found, it generates a concrete fix."

---

**[50–60s] — The Payoff**

> "Agent quality is a real competitive advantage.
> Agent Auditor takes every agent from *good* — to **great**."

---

## DELIVERY NOTES

- **Pause** naturally after "That's the gap Agent Auditor closes." — let it land.
- Say the layer numbers with a beat between each — don't rush the list.
- Drop your voice slightly on "good" and punch "**great**" at the end.
- Total word count: ~138 words. If you're running long, cut "in seconds" from Layer 1.
